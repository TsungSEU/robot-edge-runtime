# 推荐硬件配置

本文档描述 Aurora-Edge-Runtime 系统的推荐硬件配置，以获得最佳性能和用户体验。

## 计算平台

### 边缘计算模块

| 参数 | 推荐配置 | 高性能配置 |
|------|----------|------------|
| **SoC** | NVIDIA Jetson Orin NX | NVIDIA Jetson AGX Orin |
| **CPU** | ARM64 Cortex-A78AE x8 @ 2.0GHz | ARM64 Cortex-A78AE x12 @ 2.2GHz |
| **GPU** | 1024 CUDA Cores, 32 Tensor Cores | 2048 CUDA Cores, 64 Tensor Cores |
| **内存** | 16GB LPDDR5 | 64GB LPDDR5 |
| **存储** | 256GB NVMe SSD | 1TB NVMe SSD |
| **AI性能** | 100 TOPS (INT8) | 275 TOPS (INT8) |
| **功耗** | 15W-25W | 15W-60W |

### 工控机

| 品牌 | 型号 | 配置 | 适用场景 |
|------|------|------|----------|
| **NVIDIA** | Jetson Orin NX | 8核 ARM + 16GB + 256GB | 人形机器人 |
| **NVIDIA** | Jetson AGX Orin | 12核 ARM + 64GB + 1TB | 高性能场景 |
| **Intel** | NUC 12 Pro | i7-12700H + 32GB DDR5 | 实验室/研发 |
| **Advantech** | UNO-2485 | i5-1235U + 16GB + 512GB | 工业现场 |

## 机器人平台规格

### 人形机器人参考设计

参考智元机器人X2和 Google RT-2 的硬件规格：

| 参数 | 推荐规格 | 备注 |
|------|----------|------|
| **自由度** | 12-20 DOF | 每腿 6-10 DOF |
| **身高** | 1.2m - 1.7m | 可调节 |
| **体重** | < 50kg | 含电池 |
| **腿长** | 上腿 35cm + 下腿 35cm | 参考成年人体型 |
| **髋宽** | 10cm - 15cm | 稳定性考虑 |

## 电源系统

### 主电源

| 参数 | 推荐规格 |
|------|----------|
| 电池类型 | Li-ion 18650 或 Li-Po |
| 标称电压 | 24V - 48V |
| 容量 | 20Ah - 40Ah |
| 续航 | 2-4 小时 (典型工作) |
| 充电时间 | < 2 小时 |

### 电源管理

| 功能 | 推荐配置 |
|------|----------|
| BMS | 电池管理系统，支持过流/过压/温度保护 |
| DC-DC 转换 | 多路输出 (24V/12V/5V) |
| 电源监控 | 实时电压、电流、温度监测 |

## 性能预期

在推荐配置下，系统性能如下：

| 指标 | 预期值 | 说明 |
|------|--------|------|
| ONNX 推理延迟 | < 5ms | 43 维状态空间 |
| 传感器融合频率 | 100Hz | 里程计 + IMU |
| 步态检测延迟 | < 10ms | 足端检测 |
| 步态规划频率 | 50Hz | 实时轨迹生成 |
| 数据采集写入 | > 100MB/s | NVMe SSD |
